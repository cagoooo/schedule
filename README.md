# 禮堂&專科教室&IPAD平板車預約系統 v2.0.0

學校禮堂、專科教室及IPAD平板車線上預約借用系統，支援查詢與登記借用時段。

## 功能特色

### 📱 手機端月視圖排版優化 (v2.0.0)
- **Grid 網格同步** - 標題與內容完美對齊，解決邊框對應位移問題
- **RWD 格線系統** - 透過容器間距控制取代獨立邊框，讓視覺效果更一致
- **行動端適配微調** - 解決週一文字顯示不全問題，優化極小螢幕的單元格高度
- **UI 色彩一致性** - 統一行動端與桌面端的背景漸層與標籤樣式

### 🔍 搜尋體驗升級 (v1.8.0)
- **自動擴大範圍** - 自動搜尋未來 180 天內的預約，無需手動調整日期
- **關鍵字搜尋** - 支援同時搜尋「預約者姓名」與「預約理由」

### 🗓️ 月視圖優化 (v1.7.0)
- **詳細資訊顯示** - 月曆格子直接顯示預約節次與借用者
- **節次簡稱** - 自動簡化節次名稱以節省空間
- **更多提示** - 單日預約過多時自動摺疊

### 📅 預約間隔放寬 (v1.7.0)
- **半年預約** - 支援查詢與預約未來 180 天內的時段

### 📜 歷史記錄與批次預約 (v1.6.0)
- **歷史記錄查詢** - 按月份查看過去預約記錄
- **批次預約功能** - 支援一次選擇多個日期進行預約
- **日曆多選模式** - 視覺化選擇多個預約日期

### 🔍 進階搜尋功能 (v1.5.0)
- **快速搜尋** - 按預約者姓名搜尋
- **節次篩選** - 指定特定節次過濾
- **日期範圍** - 結合日期選擇器篩選
- **搜尋高亮** - 關鍵字高亮顯示
- **快速跳轉** - 點擊結果直接跳轉至該週

### 🎨 UI 繽紛優化 (v1.4.0)
- **全新配色系統** - 紫藍漸層主色調，更現代活力
- **彩色陰影效果** - 元件陰影帶有主題色彩
- **優化圓角設計** - 更柔和的視覺效果
- **預約卡片美化** - 紫色漸層背景

### 🔐 安全性強化 (v1.3.0)
- **Firestore 規則強化** - 完整欄位驗證，防止惡意資料注入
- **Rate Limiting** - 每小時/每天預約次數限制，防止濫用
- **裝置識別** - 追蹤預約來源裝置

### ✨ UI/UX 體驗優化 (v1.2.0)
- **微互動動畫** - 視圖切換淡入淡出、彈窗縮放動畫、按鈕點擊回饋
- **骨架屏 (Skeleton Loading)** - 資料載入時顯示閃爍骨架，提升感知效能

### 📅 預約管理
- **週視圖 / 月視圖** - 靈活切換檢視模式
- **多節次預約** - 支援單日多時段預約
- **重複預約** - 可設定每週重複預約
- **預約詳情** - 點擊已預約時段查看詳細資訊

### 🔐 管理員功能
- **Firebase 驗證** - 安全的管理員登入機制
- **取消預約** - 已登入管理員可取消預約
- **資料管理** - Firestore 雲端資料庫儲存

### 📊 統計功能
- **節次使用率** - 視覺化圓餅圖呈現
- **預約者排行** - 顯示預約次數最多的使用者
- **趨勢圖表** - 本月預約趨勢分析
- **匯出 CSV** - 預約資料匯出功能

### 📱 RWD 響應式設計
- **手機端優化** - 週曆垂直排列，易於上下滑動檢視
- **底部預約按鈕** - 手機端專用的大型預約按鈕
- **觸控友善** - 適合觸控螢幕操作

## 技術架構

- **前端**: HTML5, CSS3, Vanilla JavaScript
- **後端**: Firebase Firestore (資料庫)
- **驗證**: Firebase Authentication
- **部署**: Firebase Hosting / GitHub Pages

## 本地開發

```bash
# 啟動本地伺服器
npx http-server -p 8080 -c-1

# 開啟瀏覽器
http://127.0.0.1:8080
```
